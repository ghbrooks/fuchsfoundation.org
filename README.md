# fuchsfoundation.org

Website for the Fuchs Foundation (Charity No. 271084) — a charity that inspires science and geography teachers through polar expeditions, encouraging leadership in education and providing free teaching resources.

## Structure

- **Pages**: `index.html`, `foundation.html`, `expeditions.html`, `resources.html`, `scholars.html`, `give.html`, `contact.html`, `privacy.html`, `terms.html`, `sitemap.html`
- **Expedition pages**: `expeditions/` — individual expedition write-ups (Antarctic 2007, Arctic 2009, Antarctic 2010, Iceland 2014, Greenland 2016)
- **Resource pages**: `resources/` — lesson plans and scientific reports
- **Styles**: `css/styles.css`
- **Scripts**: `js/main.js`
- **Images**: `images/`, `uploads/`
- **Legacy pages**: `page/` — original CMS-style routes preserved for redirects

## Development

Static HTML/CSS/JS site. No build step required — open `index.html` in a browser or serve with any static file server.
